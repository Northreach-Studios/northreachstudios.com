Accessibility Statement

Last updated: 30 May 2026

Our Commitment

Northreach Studios wants this website to be usable by as many people as possible. We aim to meet the Web Content Accessibility Guidelines version 2.2 at level AA where practical for this small static site.

What The Site Supports

  • Responsive layouts for mobile, tablet, and desktop screens.
  • Keyboard-visible focus styles for links and controls.
  • A skip link to move directly to the main content.
  • Semantic HTML landmarks and headings.
  • Alternative text for meaningful images.
  • No required time limits, autoplay audio, or flashing content.

Known Limitations

This website has not yet had an independent accessibility audit or user testing with disabled users. We will review this again as the studio and website grow.

Feedback And Contact

If you find an accessibility issue or need information in another format, email support@northreachstudios.com.

Please include the page URL, the issue you found, and the assistive technology or browser you were using if relevant.

Review

We will review this statement when the website changes materially or when new interactive features are added.
