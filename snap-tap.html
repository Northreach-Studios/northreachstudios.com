Snap Tap

Snap Tap is a fast, replayable reflex game built for quick sessions on mobile, with a clear arcade-style loop focused on reaction speed, precision, and repeat play.

In Development Back to Games

Project Status

Core gameplay is being stabilised and polished.

Overview

Designed for quick sessions and clean skill expression.

Snap Tap is built around immediate readability and satisfying replayability. The aim is a mobile game that feels easy to pick up, quick to understand, and rewarding to improve at through sharper reactions and better consistency from run to run.

The design direction stays intentionally lean. Rather than adding friction or unnecessary complexity, the focus is on making each session responsive, legible, and satisfying from the first tap through game over.

Current Focus

What the team is working on now.

  • Removing temporary test functionality as core systems settle.
  • Implementing a bottom timer bar UI to show remaining session time.
  • Continuing gameplay polish and presentation improvements ahead of release.
